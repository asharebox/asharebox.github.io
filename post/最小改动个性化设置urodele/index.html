Urodele作为Astro项目，样式处理方式不同。基于实际Astro项目结构的最小改动方案。

---

🎯 Urodele 正确文件结构 + 最小改动指南

根据Astro标准结构和Urodele实际部署情况，**正确的文件位置**应该是：

```

your-blog.github.io/

├── src/

│ ├── components/ # React/Vue组件（Header含Login按钮）

│ ├── layouts/

│ │ └── Layout.astro # 主布局（在这里导入全局样式）

│ ├── pages/

│ │ ├── index.astro # 首页

│ │ └── post/

│ │ ├── [...id].astro # 文章页（含Edit按钮）

│ │ └── edit.astro # 编辑页

│ └── styles/ # ★ 需要新建这个文件夹

│ └── global.css # ★ 新建：全局样式文件

├── urodele.config.ts

└── astro.config.mjs

```

Step 2: 在布局中导入样式（修改现有文件）

**修改 src/layouts/Layout.astro**，在 <head> 中添加样式导入：

```astro

---

// 保留所有原有 imports

import '../styles/global.css'; // ★ 新增：导入全局样式

const { title } = Astro.props;

---

<!DOCTYPE html>

<html lang="zh-CN">

<head>

<meta charset="UTF-8" />

<meta name="viewport" content="width=device-width, initial-scale=1.0" />

<title>{title}</title>

<!-- 保留所有原有 head 内容 -->

</head>

<body>

<!-- 保留原有 slot，组件会自动渲染 -->

<slot />

</body>

</html>

```

**注意**：如果Layout.astro中已经有 <style> 标签或导入其他CSS，**不要删除**，只需在上面添加一行 import。

## 如果 src/styles/ 文件夹不存在

**手动创建文件夹和文件**：

```bash

# 在项目根目录执行

mkdir -p src/styles

touch src/styles/global.css

```

然后复制上面的CSS代码进去。

## 如果 Layout.astro 导入样式后没效果

检查是否有 **CSS优先级** 问题，在 global.css 顶部添加：

```css

/* 提高优先级，覆盖默认样式 */

:root {

--bg-primary: #0a2e2e !important;

--text-primary: #e0e0e0 !important;

/* ... 其他变量 */

}

```

或者在Layout.astro中确保你的import是**最后**一个样式导入（后导入的优先级高）。

## 更改主题颜色的最小改动步骤

### 只需修改 2 个文件

### 文件 1uno.config.ts（颜色定义）

```typescript

// 修改这里的 colors 对象

const themes = {

dark: {

colors: {

// 背景

bg: "#1e2030", // ← 改这里：主背景

// 文字

text: "#c8d3f5", // ← 改这里：主文字

comment: "#565f89", // ← 改这里：次要/灰色文字

// 强调/链接

primary: "#4a9eff", // ← 改这里：主按钮

function: "#7aa2f7", // ← 改这里：链接颜色

// 边框

border: "#292e42", // ← 改这里：边框

// 其他（按需修改）

gray: "#2a2e45",

selection: "#3b4261",

},

},

};

```

### 文件 2index.scss（硬编码颜色）

搜索并替换这些颜色值：

| 用途 | 当前值 | 改为新值 |

|------|--------|---------|

| header 背景 | #1e2030 | 新背景色 |

| header 边框 | #292e42 | 新边框色 |

| 按钮背景 | rgba(74, 158, 255, 0.1) | 新主色透明 |

| 按钮文字 | #7aa2f7 | 新主色 |

| 悬停背景 | rgba(74, 158, 255, 0.15) | 新主色透明 |

| 滚动条 | rgba(59, 66, 97, 0.5) | 新灰色 |

| hr 边框 | #292e42 | 新边框色 |

| 选中文字背景 | rgba(74, 158, 255, 0.3) | 新主色透明 |

### 快速替换命令（示例）

如果你想从**深蓝**改为**深绿**：

```bash

# 在 index.scss 中替换

sed -i 's/#1e2030/#0a2e2e/g' src/index.scss

sed -i 's/#292e42/#1a4a4a/g' src/index.scss

sed -i 's/#7aa2f7/#6b9999/g' src/index.scss

sed -i 's/#c8d3f5/#e0e0e0/g' src/index.scss

sed -i 's/#565f89/#5c8a8a/g' src/index.scss

sed -i 's/rgba(74, 158, 255/rgba(107, 153, 153/g' src/index.scss

sed -i 's/rgba(59, 66, 97/rgba(92, 138, 138/g' src/index.scss

```

同时在 uno.config.ts 中修改对应颜色值。
