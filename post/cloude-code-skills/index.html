10 cloude code skills

1. Claudeception —— 自动进化技能

一句话介绍：干活时自动总结经验，保存成 Skill

这是我自己用的最多的一个。它会在 Claude Code 干活的时候，自动检测有没有值得沉淀的经验——比如你连续几次都先查某个 API 文档，它就会自动创建一个"获取这个文档"的新技能。

相当于你每用一次 Claude Code，它都在悄悄帮你积累经验，后面同类问题直接调用技能就行。

安装：npx skills add claudeception

2. skills.sh —— 找最火的 Skills

一句话介绍：Vercel 出的 Skills 聚合站，按安装量排名

不知道装什么好？直接看这个。skills.sh 会按 24 小时内的安装量排序，让你知道现在最火的是什么。

相当于一个"Skills 热门榜"，每天刷一刷，发现新神器。

安装：直接访问 skills.sh

Top 10

3. Remotion —— 免费做 JS 动画

一句话介绍：用代码做动画，还能做 3D

Remotion 原本是做视频动画的，配合这个 Skill，Claude Code 可以直接帮你生成 JS 动画。比如做个"如何安装 Claude Code"的文字讲解视频，用在 PPT 里绝杀。

关键是免费的，比那些付费动画工具香多了。

安装：npx skills add remotion-dev/skills

4. Pencil —— 无限画布版 Claude Code

一句话介绍：兼容 Figma，自带设计规范

Pencil 可以理解为"无限画布版"的 Claude Code，兼容 Figma。它自带了设计规范和示例风格，生成的界面可以直接用。

适合做产品原型、UI 演示，或者给客户展示交互方案。

安装：npx skills add pencil-dev/skills

5. Skill Vision Control —— 更新不丢本地修改

一句话介绍：下载新版本时保留旧的，对比后再决定

用 Skills 的朋友都知道，很多 Skill 在本地会修改着用。但官方更新后，本地修改会被覆盖。

这个 Skill 解决了这个问题：下载新版本时保留旧版，对比后再决定用哪个。算是 Skills 管理必备了。

安装：npx skills add xiaoerzhan/skill-vision-control

6. NanoBanana-PPT-Skills —— 动态 PPT 一键生成

一句话介绍：分析文档做 PPT 大纲，用可灵做动画，一键合成视频

这是最近跑下来最惊艳的一个。流程是这样的：

  • 分析你的文档，生成 PPT 大纲

  • 用 Banana2 生成图片

  • 用可灵做页面过渡动画

  • 一键合成包含所有转场的 PPT 视频

做分享、演示、教学视频，特别省事。

安装：npx skills add op7418/NanoBanana-PPT-Skills

7. notebooklm-skill + Theme Factory —— 批量生成风格化 PPT

一句话介绍：一次做 15 张，100 页都能保持风格统一

notebooklm-skill 本身可以生成 PPT，但最多 20 页。配合 Theme Factory 这个"学风格"的 Skill，可以突破限制：

  • 先让 AI 学你想要的 PPT 风格（配色、字体）

  • 然后生成 PPT，一次 15 张

  • 循环生成，100 页都能保持风格统一

做品牌演示、产品介绍，特别实用。

安装：

  • npx skills add PleasePrompto/notebooklm-skill

  • npx skills add Theme Factory

8. homunculus —— 自动创建技能

一句话介绍：连续做某个请求前都会先看文档？它会自动创建新技能

比如你连续三四次做某个请求前都会先看 API 文档，homunculus 会自动创建一个"自动获取文档"的新技能。

相当于你不用自己写 Skill 了，AI 观察你的习惯后自动生成。

安装：npx skills add humanplane/homunculus

9. skill-from-masters —— 专家方法论变技能

一句话介绍：新建技能时，网络搜索领域专家的方法论

这个更猛：当你创建一个新技能的时候，它会自动去网络搜索这个领域的专家方法论，或者找高赞的 GitHub 项目，转化为新技能。

相当于一个"智能技能生成器"，站在巨人的肩膀上。

安装：npx skills add skill-from-masters

10. Document Processing —— 带格式生成 Office 文档

一句话介绍：带格式、带公式生成 Word/Excel/PPT/PDF

这个是压轴的。Claude Code 配合这个 Skill，可以带格式、带公式生成各种 Office 文档。

之前你可能需要手动复制粘贴排版，现在直接输出可编辑的 Word/Excel/PPT，科研党、办公党狂喜。

安装：npx skills add ComposioHQ/awesome-claude-skills
