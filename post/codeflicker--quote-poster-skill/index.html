Quote Poster Generator

Quote Poster Generator 是一个专门用于将用户输入的文本转换为精美金句海报的 skill。

核心功能

  1. 文本分析与提取

    • 提取用户输入的核心金句/标语作为主标题

    • 识别作者或出处（如果有）

    • 分析文本的基调，以决定海报的视觉风格（如：科技感、极简风、文艺风等）

  2. 副标题自动生成

    • 必须生成副标题：根据主标题内容，自动生成一个副标题

    • 副标题作用：解释主标题含义、扩展主题思想、或提供行动建议

    • 字数限制：副标题必须在15个汉字以内

    • 风格要求：简洁有力，与主标题形成呼应

    • 示例：

      • 主标题："你不要给我诉苦，我要结果" → 副标题："执行力胜过千言万语"

      • 主标题："在这个时代你还在考虑钱就落后了" → 副标题："用AI杠杆放大你的产能"

      • 主标题："选择比努力更重要" → 副标题："方向对了，慢一点也是快"

  3. 海报设计参数确定

    • 颜色主题: 背景色、文字颜色、高亮颜色

    • 排版结构: 大标题、副标题、正文、作者署名

    • 视觉元素: 图标、线条、几何形状等装饰

  4. 生成海报代码

    • 使用 HTML/CSS 结合 html2canvas 或通过 Python 的 Pillow 库来生成海报

    • 也可以生成 Mermaid 语法或 SVG 代码来快速呈现效果

    • 默认建议使用 HTML/CSS 生成现代、高质量的海报效果，然后通过浏览器截图或打印为 PDF/图片

工作流程

当用户触发此 skill 时，请按照以下步骤执行：

第一步：分析用户输入

如果用户只是一句话，直接将其作为海报主体文字。 如果用户输入较长，提取出最具冲击力的一句话作为大标题（标语），其余作为副标题或说明文字。

第二步：确认设计风格（可选）

如果用户没有指定风格，默认使用现代、简洁、对比强烈的风格（如黑色背景+亮黄色/白色文字，类似知识分享类海报）。 如果需要，可以向用户确认："我为您构思了深色极简科技风，包含大字高亮设计，是否可以直接为您生成？"

第三步：生成代码并渲染

编写高质量的 HTML 和内联 CSS 代码。 要求：

  • 使用现代字体（如 system-ui, -apple-system, PingFang SC, Microsoft YaHei）

  • 良好的留白（Padding/Margin）

  • 强烈的视觉层次感（大字号主标题，高亮关键词）

  • 合适的背景色和文字颜色对比度

提供代码后，指导用户如何查看效果（例如：保存为 .html 文件在浏览器中打开，或者如果我们在具有预览能力的环境中，直接渲染）。

最佳实践提示

  • 高亮关键词：对于一句话，找出其中的动词或核心名词，使用不同的颜色（如亮黄色）进行高亮，增加视觉冲击力。

  • 排版对齐：通常左对齐或居中对齐看起来最专业。

  • 添加装饰：可以添加简单的引号图标 ""，或者上下分隔线，增加海报感。

  • 副标题设计：

    • 副标题字号应小于主标题，通常使用16px左右

    • 颜色使用灰色（如 #888）以区分层级

    • 副标题应对主标题进行解释、扩展或补充，提供更深层的思考

    • 严格控制字数在15字以内，保持简洁有力

</style>
</head>
