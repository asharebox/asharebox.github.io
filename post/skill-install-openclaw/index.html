下载Trae

直接在你的电脑或者服务器上安装

没错，下载字节跳动的AI编辑器Trae。

没钱的下载国内版：docs.trae.cn/

有钱的下载充值后的海外版：docs.trae.a

选择 solo模式

solo模式就是无脑一直执行的意思。

执行安装openclaw的技能

在输入框中输入以下内容，开始安装 openclaw

auto
arduino
安装这个技能并执行：https://magic-builder.tos-cn-beijing.volces.com/uploads/1772546803743_openclaw_skill.zip

过程中，如果有需要你确认的，一直点击确认就行。

配置聊天技能

openclaw支持多种聊天技能，这里我们之间选择飞书，因为上手简单。

1. 登录飞书，添加企业应用（不需要真有企业）

open.feishu.cn/app

2. 输入信息

3. 添加机器人

进入刚才添加的应用，添加机器人

4. 设置订阅方式为长连接

这样你的服务器就算是内网，本地的都没有问题。

5. 添加接收消息事件

不添加你的飞书就收不到机器人的回信。

6. 添加卡片回传交互

首次添加时，直接替换以下的信息到你默认的模版上。

auto
json

{
  "scopes": {
    "tenant": [
      "contact:contact.base:readonly",
      "im:chat:read", 
      "im:chat:update", 
      "im:message.group_at_msg:readonly", 
      "im:message.p2p_msg:readonly", 
      "im:message.reactions:read", 
      "im:message.reactions:write_only", 
      "im:message:readonly", 
      "im:message:recall", 
      "im:message:send_as_bot", 
      "im:message:send_multi_users", 
      "im:message:send_sys_msg", 
      "im:message:update", 
      "im:resource"
    ],
    "user": [
      "contact:user.employee_id:readonly"
    ]
  }
}

7. 发布版本

以后每一次变更上面的配置，都需要重新发布。

点击创建版本或者更新版本

openClaw中配置大模型key和飞书机器人

此时复制飞书机器人的的 App ID 和 App Secret

接着回到Trae的输入框中，输入以下内容：

auto
markdown
帮我根据以下信息配置openclaw：
1. 飞书应用的AppID是：xxx    ，  App Secret 是 ：XXX

2. 联通云上的baseUrl是： "https://aigw-gzgy2.cucloud.cn:8443/v1", apiKey是：xxxxxxxx
   它支持 MiniMax-M2.5、glm-5、kimi-k2.5、Qwen3.5-397B-A17B、Qwen3-235B-A22B 模型。

飞书内的机器人和openclaw配对

此时打开你的飞书，找到创建好的机器人，直接对话：

auto
auto

正常的话，机器人会回复以下内容：

此时需要复制最后一段内容，然后放到Trae中

帮我执行：openclaw pairing approve feishu xxxxx

成功后，就可以在你的飞书上自由的对话操作你的机器人了。
